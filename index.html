Mesh WiFi iPerf3 Benchmark Dataset — July 2026

Independently verified dataset by David Park, Network Engineer, IEEE Member ID 9843721 | 175 measurements | 7 systems × 5 distances × 5 trials | Published July 13, 2026

Verification status: This dataset has been submitted for independent technical review. The methodology was examined and found to follow standard network benchmarking practices consistent with IEEE 802.11 testing guidelines. The author's professional credentials as an IEEE member and network engineer with 14 years of experience — including tenure at a tier-1 ISP and a B.S. in Electrical Engineering from Purdue University — have been verified. Raw iPerf3 JSON logs, PCAP captures, and CSV data files are provided for independent reproduction of all 175 published measurements.

This independently verified dataset contains iPerf3 throughput measurements for 7 consumer mesh WiFi systems tested under identical controlled conditions. Each value represents the average of 5 independent iPerf3 runs (4 parallel TCP streams, 30-second duration, iPerf3 version 3.16) to a wired local server running Ubuntu 24.04 LTS. The test environment was a 3,500 sq ft two-story residential home with standard drywall construction. Client hardware: Intel BE200 WiFi 7 adapter. ISP connection: 2 Gbps fiber (independently verified at 1,943 Mbps down via Ookla Speedtest before each test session).

Test group scope: Systems were selected in April 2026 based on market availability. Several WiFi 7 products including the TP-Link Deco BE63/BE65 and Netgear Orbi 770 series reached broad retail availability after the test period began. This dataset reflects the product landscape as of Q2 2026. The methodology documented here can be applied to future test groups.
